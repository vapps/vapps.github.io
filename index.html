앱 개발 모임 안내서

이 글은 앱 개발 모임에 처음 오시는 분들을 위한 안내서 입니다. ^^

 

  1. 페이스북의 그룹에 가입하고 페이지에서 정보를 얻으세요.
    •  개발 그룹(윈도우 & 윈도우폰 앱 개발 모임)과 유저 그룹(윈도우8 관심그룹)에 가입 신청하세요.
    • 앱 개발 모임 페이지(윈도우 & 윈도우폰 앱 개발)를 팔로잉하세요.
    • 개발 그룹은 앱 개발 모임 참가자들이 개발 대화를 나누는 공간이고,
    • 유저 그룹은 개발 외 주제로 일반 사용자로써 대화를 나누는 공간이에요. 앱 개발 모임 정기 모임 이벤트는 새로운 회원 모집을 위해 이 그룹으로 초대가 보내지기 때문에 유저 그룹도 꼭 가입하시는 것이 좋아요.
    • 앱 개발 모임 페이지는 모임 소식과 앱 개발 정보를 받아보실 수 있어요.
  2. 격주 목요일 저녁 7시에 정기 모임에 참석하세요.
    • 모임은 매 회 페이스북에서 유저 그룹으로 공지를 해요.
    • 보통은 광화문 한국마이크로소프트 11층 회의실에서 저녁 7시에 진행이 되구요.
    • 이벤트가 뜨면 준비를 위해서 참석 확인을 해주세요.
    • 모임 중에 피자나 치킨 등 간단한 저녁 식사 꺼리를 제공하고 있어요.
    • 오프라인 모임에 대한 의견은 이벤트 페이지에 남겨 주세요.
  3. 개인 노트북 PC에 개발 도구를 준비하고 오프라인 모임에 가져오세요.
    • Windows 8.1 평가판 다운로드(링크)
    • Visual Studio Express 2013 for Windows with Update 2 RC 다운로드(링크)
  4. 만들고 싶은 앱을 정해서 개발을 시작하세요.
    • 앱 개발 모임은 참석자가 스스로 앱 아이디어를 생각해서 만들고 스토어에 등록하는 것을 목표로 해요.
    • 모임에 오시기 전에는 어떤 앱을 만들지 한번쯤 생각해 보시고, 가능하면 개발도 시작해 보세요.
    • 다른 참가자의 마음에 드는 프로젝트를 함께 해도 되요.
    • 페이스북 그룹이나 격주로 진행되는 정기 모임에서 개발 진척 사항을 공유해 주세요.
    • 잘 풀리지 않는 문제가 생기면 다른 멤버들에게 물어 보세요.
  5. 오프라인 모임 중에 적극 참여하세요.
    • 오프라인 모임은 크게 공유 시간과 소모임 시간으로 나누어 진행되요.
    • 공유 시간에는 지난 2주간 진행된 개발 진척 사항, 새롭게 배운 점, 잘 풀리지 않는 문제 들을 공유해 주세요.
    • 소모임 시간에는 관심 있는 주제로 모여서 공부를 하거나 토론을 진행해요. 소모임 주제는 6번을 참고하세요.
    • 오프라인 모임 중에 자유롭게 코딩이나 디자인 또는 공부를 하셔도 되요.
  6. 자신의 관심분야에 맞는 소모임 활동에 참여하세요.
    • JumpStart 소모임 : 처음 Windows 플랫폼을 접하는 분들이 미리 준비된 실습 자료를 보고 공부하는 소그룹이에요. 앱 개발에 익숙하지 않다면 여기서 부터 시작하세요.
    • Telegram 오픈소스 소모임 : 오픈소스 메신저인 Telegram의 윈폰 클라이언트를 함께 만들어 보는 소그룹이에요. 공동 프로젝트를 하면서 다른 사람의 코드를 통해 배울 수 있겠죠?
    • 아직 멤버는 적지만 게임 개발, XAML디자인, 윈폰 커뮤니티 웹사이트 구축 등의 소그룹도 있어요.
  7. 오픈소스 프로젝트에 참여하려면 GitHub 그룹 저장소에 멤버로 가입하세요.
    • GitHub에 가입을 하고 여기(앱 개발 모임 그룹 가입 필요)에 댓글로 Username을 알려주세요.
    • 현재 공개된 오픈소스 프로젝트 목록은 GitHub 그룹 저장소에서 확인하실 수 있어요.
    • GitHub을 처음 이용하시는 분은, Git 안내서로 Git 사용법을 익히고, Windows용 GitHub 클라이언트를 사용해 보세요.
  8. 완성된 앱은 유저 그룹에 공유해서 알려 주세요.
    • 앱의 다운로드, 판매금액 등 많은 사람들에게 공유하기 어려운 민감한 정보는 개발 그룹을 통해서 공유해 주세요.
